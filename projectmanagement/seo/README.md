# RELENTLESS SEO Plan

Last updated: May 21, 2026

## Scope

This folder tracks the first-month SEO rollout for RELENTLESS Mouth Guards, with a local focus on Tacoma and the greater Seattle area.

Primary goals:

- Make the site crawlable, understandable, and clean in search results.
- Build local relevance for Tacoma, Seattle, and South Sound contact-sport searches.
- Create landing pages for high-intent mouthguard buyers.
- Track what was implemented, what is pending, and what should be measured.

## Files

- `MONTH-1.md` - first 30-day SEO roadmap.
- `LOCAL-SEO.md` - Tacoma and greater Seattle keyword/page plan.
- `KEYWORDS.md` - keyword list and page mapping.
- `LANDING-PAGES.md` - local and sport landing page build plan.
- `BLOG-CONTENT.md` - blog archive, posts created, and next blog queue.
- `CHECKLIST.md` - implementation checklist for technical, content, local, and tracking tasks.

## Priority

1. Technical SEO foundation: metadata, canonical URLs, sitemap, robots, structured data, social previews.
2. Local SEO foundation: Tacoma and greater Seattle wording, contact page updates, local landing pages.
3. Buyer-intent content: product, sport, team/gym, and process pages.
4. Measurement: Search Console, Bing Webmaster Tools, conversion events, and monthly review.
